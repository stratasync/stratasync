These guides walk through common patterns and advanced features with complete, runnable code examples.

## Guides

- [Offline-first patterns](/docs/guides/offline-first) -- How the outbox, optimistic updates, and reconnection flow work together to deliver a seamless offline experience.
- [Collaborative editing with Yjs](/docs/guides/collaborative-editing) -- Set up real-time, multi-user editing with CRDTs, presence tracking, and rich-text editor integration.
- [Conflict resolution](/docs/guides/conflict-resolution) -- Understand the server-sequenced model, rebase strategies, and field-level conflict detection.
- [Server-side rendering and bootstrap](/docs/guides/ssr-bootstrap) -- Prefetch data on the server, serialize snapshots, and hydrate the client for instant first paint.
- [Model relationships](/docs/guides/model-relationships) -- Define references, collections, and back-references between models with lazy and eager loading.
- [Load strategies](/docs/guides/load-strategies) -- Control when and how model data is fetched from the server with instant, lazy, partial, and other strategies.
