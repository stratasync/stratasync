We designed Strata Sync as a layered system with strict dependency boundaries. Each layer handles a specific concern, and framework-specific integrations live in adapter packages at the edges.

## In this section

- [Overview](/docs/architecture/overview) -- The layered architecture, package dependency graph, and the role of each layer.
- [Sync protocol](/docs/architecture/sync-protocol) -- How the server-sequenced sync model works: the monotonic sync log, delta streaming, bootstrapping, and catch-up.
- [Data flow](/docs/architecture/data-flow) -- Trace a mutation from user input through the outbox to the server and back, and understand how server deltas flow to all connected clients.
