Install the packages for your stack.

## Requirements

- **Node.js** >= 18
- **TypeScript** 5.9+ with `experimentalDecorators` enabled
- **React** 18 or 19
- **Next.js** 14 or 15 (if using `@stratasync/next`)

## Packages

Pick the row that matches your stack and run the install command.

| Use case                   | Command                                                                                                                                                     |
| -------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Sync client (no framework) | `npm install @stratasync/core @stratasync/client @stratasync/mobx @stratasync/storage-idb @stratasync/transport-graphql`                                    |
| React app                  | `npm install @stratasync/core @stratasync/client @stratasync/react @stratasync/mobx @stratasync/storage-idb @stratasync/transport-graphql`                  |
| Next.js app                | `npm install @stratasync/core @stratasync/client @stratasync/react @stratasync/next @stratasync/mobx @stratasync/storage-idb @stratasync/transport-graphql` |
| Collaborative editing      | Add `@stratasync/y-doc` to any of the above commands                                                                                                        |

## Peer dependencies

Three packages require peer dependencies that your application must provide.

| Package             | Peer dependencies                         |
| ------------------- | ----------------------------------------- |
| `@stratasync/react` | `react ^18 \|\| ^19`, `yjs ^13.6`         |
| `@stratasync/next`  | `next ^14 \|\| ^15`, `react ^18 \|\| ^19` |
| `@stratasync/mobx`  | `mobx ^6.0`                               |

Other packages bundle their dependencies.

```bash
npm install react mobx yjs
```

## TypeScript configuration

Enable experimental decorators so Strata Sync can use them for model schema definitions.

```json
{
  "compilerOptions": {
    "experimentalDecorators": true,
    "emitDecoratorMetadata": false,
    "strict": true,
    "target": "ES2022",
    "module": "ESNext",
    "moduleResolution": "bundler"
  }
}
```

You don't need `emitDecoratorMetadata` — Strata Sync decorators work through runtime registration with the `ModelRegistry`.
