Strata Sync is a local-first, server-sequenced sync engine for TypeScript, React, and Next.js that enables instant, offline-capable applications with real-time collaboration and built-in undo/redo.

## What is Strata Sync?

Strata Sync implements a server-sequenced sync architecture. It provides:

- **Local-first data** — All reads are instant from a local replica
- **Server-sequenced sync** — A monotonic sync log ensures consistency
- **Observable models** — MobX-powered reactive model instances
- **Offline support** — Outbox-based mutation queue with automatic retry
- **Real-time collaboration** — Yjs CRDT integration for rich-text editing
- **Undo/redo** — Transaction-based undo with full history
